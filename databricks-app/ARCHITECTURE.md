## Key Interfaces

### 1. UserContext
```python
class UserContext(BaseModel):
    user_id: str              # e.g., "user@databricks.com"
    email: str                # e.g., "user@databricks.com"
    username: str             # e.g., "Jane Doe"
    ip_address: Optional[str] # e.g., "192.168.1.1"
    
    @property
    def session_key(self) -> str:
        return f"session:{self.user_id}"
```

### 2. Settings
```python
class Settings(BaseSettings):
    environment: Literal["development", "production"]
    log_level: str                    # "INFO", "DEBUG", etc.
    log_format: Literal["json", "text"]
    serve_frontend: bool              # True in production
    frontend_dist_path: Path          # Path to frontend/dist/
    
    @property
    def is_production(self) -> bool
    
    @property
    def is_development(self) -> bool
```
